> **适用业务**：定制礼品服饰（印字 / 印花 / 刺绣、情侣、家庭、宠物、节日款等）  
> **与当前代码项目无关**：本文档为独立运营方案，不绑定 `mall_front_web` 建材商城。

## 国内仓发国外 — 可以吗？

**可以，和本指南不冲突。**

「不铺货」在这里指：**不在美国/欧洲等地租仓、大量备现货**。  
**国内仓接单 → 定制生产 → 国际小包发出**，完全 OK，且比纯甩单代发更适合服饰（质检、换包装、控出库）。

| 发货方式 | 是否算海外铺货 | 说明 |
|----------|----------------|------|
| 美/欧海外仓大量备货 | ✅ 算 | 资金压在当地 |
| **国内仓定制后发国际** | ❌ 不算 | 你的主模式 |
| 工厂一件代发（长尾 SKU） | ❌ 不算 | 测款、补色补码 |
| 国内仓 + 代发混合 | ❌ 不算 | **推荐** |

```
独立站下单
    ├─ 定制款 → 国内仓/工厂：排版 → 生产 3～5 工作日 → 交物流 → 英美澳 7～15 工作日
    └─ 现货款/测款 → 代发或国内仓直发
```

## 和「铺货模式」差在哪

| 维度 | 海外仓铺货 | 你的模式（国内定制 + 独立站） |
|------|------------|------------------------------|
| 资金 | 压海外库存 | 压 **广告 + 少量国内备料 + 生产周转** |
| 选国 | 仓储、关税 | **签收率、ROAS、拒付、生产+物流总时效** |
| 渠道 | Amazon FBA | **独立站投流**（可对标本品站） |
| 退款 | 标品 30 天退 | **定制品：非质量问题不退**（见附录文案） |
| 旺季 | 全年 | **情人节、母亲节、圣诞等强季节** |

## 阶段 0：链路 + 独立站（3～6 周）

### 0.1 选品（定制服饰，第 1 周）

| 类型 | 建议 |
|------|------|
| 情侣 / 纪念日卫衣、T 恤 | ✅ 全年，情人节前后爆发 |
| 家庭 / Mom Dad / 宠物头像衫 | ✅ 母亲节、父亲节、圣诞 |
| 毕业 / 团队定制 | ✅ 美国 5～6 月 |
| 复杂全定制礼盒 + 超重外套 | ⚠️ 运费与退货压力大，后置 |

**输出**：5～10 个 **可预览定制** 的爆款模板（名字、日期、图案位）。

### 0.2 国内仓 + 代发（第 1～2 周）

**国内仓 / 自有产线**

- [ ] 定制 SLA：**出图确认 → 生产 3～5 工作日 → 24h 内交物流**
- [ ] 尺码表（US/UK 尺码对照）、印花位置示意图
- [ ] 库存：空白衫 + 热门色码安全库存
- [ ] 测一单全流程（录屏）：下单 → 生产 → 云途/4PX → 签收

**代发（补款）**

- [ ] 1～2 家 POD/工厂，仅用于测款色
- [ ] SKU 标记：`国内定制` / `代发`

### 0.3 独立站（第 2～3 周）

| 项 | 要求 |
|----|------|
| 平台 | Shopify / 店匠 / WooCommerce / 自建站 |
| 必备页 | Shipping、**Refund（定制品）**、Privacy、Contact |
| 定制能力 | 选图/填字 → **预览 mockup** → 加购（降低纠纷） |
| 支付 | Stripe + PayPal（美国必备） |
| 追踪 | Meta Pixel + GA4 + CAPI |

### 0.4 物流文案（定制品专用，必写）

分两段写清，不要只学 WaveBeamPro 的 6～12 天：

> **Production:** 3–5 business days after order approval.  
> **Shipping:** 7–15 business days internationally after dispatch.  
> **Total:** typically 10–20 business days.

### 0.5 合规（轻量）

| 市场 | 要点 |
|------|------|
| 美国 | 禁侵权 IP、影视角色；关注销售税阈值；拒付处理流程 |
| 英国 | GDPR 隐私页；清晰退货政策 |
| 欧盟 | 有量再做 VAT/OSS |

### 0.6 阶段 0 完成标准

- [ ] 独立站可下单并完成定制预览
- [ ] 至少 **1 单** 国内生产 + 国际签收
- [ ] 退款政策页已上线（定制品版）

## 阶段 1：独立站 + 欧美（第 2～12 月）

### 1.1 国家顺序（定制礼品服饰 — 结合 WaveBeamPro 与品类）

| 顺序 | 国家 | 说明 |
|------|------|------|
| **①** | **美国** | 定制礼品最大市场；WaveBeamPro **推断首发**也是美国（USD、对美物流叙事） |
| **②** | **英国** | 送礼文化强；素材可微调英式英语 |
| **③** | **加拿大 / 澳大利亚** | 复制盈利款 |
| ④ | 德国等 | 有德语产能再做 |

**第一年建议：美国 + 英国**（2 国做深），不要同时开 10 国广告。

> 注：通用代发指南常写「英国先测、美国后上」；**定制礼品更建议美国优先**，与 WaveBeamPro 路径一致。

### 1.2 流量顺序

```
第 2～3 月   Meta 美国测款（专题落地页，非整站堆砌）
第 3～4 月   加 Google PMax / 搜索品牌词
第 4～6 月   TikTok 短视频（开箱、送礼反应）
第 5 月+     英国复制美国盈利款
第 6 月+     邮件弃购、再营销
节日前 45～60 天   加大预算（见附录节日表）
```

### 1.3 第 2～4 月：美国测款

| 周 | 动作 |
|----|------|
| W1 | 上线 3 个定制场景页（情侣 / 妈妈 / 宠物） |
| W2 | Meta $50～80/天，1 _campaign 3  ad set（兴趣：gifts, personalized） |
| W3 | 砍 CPA 差的款，加预算有加购的 |
| W4 | 复盘：支付率、生产是否延误、首波拒付 |

### 1.4 第 5～8 月：放量 + 英国

- 美国：放大 ROAS &gt; 1.2～1.3 的 SKU/素材
- 英国：GBP 定价、运费模板、同一套 mockup
- 关注：**印错字纠纷**、物流断更、节日前产能

### 1.5 第 9～12 月：加澳/加 + 旺季

- 只复制 **已盈利 5～8 个模板**
- 圣诞、黑五前 **提前排产**，避免 12 月爆仓延误

### 1.6 阶段 1 完成标准

| 指标 | 目标 |
|------|------|
| 盈利定制模板 | ≥ 3 个 |
| 生产+交物流 | 承诺时效内 ≥ 90% |
| 退款率 | &lt; 8%（定制品高于标品正常） |
| 稳定盈利国 | ≥ 2（建议美+英） |

## 阶段 2：中东（可选，第 9 月+）

定制服饰（图案、服饰禁忌、COD、物流慢）→ **默认不投**。  
仅当：有海湾专线 + 签收率 &gt; 85% + 图案已本地化审核，再小预算试阿联酋英语人群。

## 定制礼品服饰：哪些国家更好卖

### 第一档（主攻）

| 国家 | 原因 |
|------|------|
| **美国** | 节日多、个性化付费意愿强、广告池最大 |
| **英国** | 圣诞、Mother's Day、送礼心智强 |
| **澳大利亚 / 加拿大** | 英语复制美式素材 |

### 第二档（盈利后再开）

德国、法国、荷兰 — 需语言与合规。

### 不建议起步

中东、东南亚低价国、日韩（除非重度本地化）。

### 按场景

| 场景 | 更突出的国家 |
|------|--------------|
| 情侣 / 纪念日 | 美、英、澳 |
| 妈妈 / 爸爸 / 家庭 | 美、英、加 |
| 宠物定制 | 美、英、澳 |
| 毕业团队 | 美、加（5～6 月） |

### 2. 「铺很多国家」到底是什么意思

**不是** 在每个国家海外备货。

```
┌──────────────────────────────────────┐
│  一个域名 wavebeampro.com             │
│  一套英文落地页 + Shopify Markets 式结账 │
│  FREE SHIPPING 全球叙事               │
└──────────────────────────────────────┘
         ↑ 分国家/地区投 Meta 广告
         ↑ 结账时再算运费/税（如有）
```
